# Dealinstinct V2 Description

Dealinstinct V2 is a Next.js repository in the Dealinstinct workspace group.

It belongs to Profound Projects and is tracked by the Sentient OS workspace registry.

## Role In Profound Projects

This repo provides a V2 Dealinstinct application surface. It should be treated as an active product repo with review-gated product changes.

## Sapient Connection

Sapient memory lives in:

- `Description.md`
- `Architecture.md`
- `Decisions.md`
- `Roadmap.md`

These files describe identity, architecture, durable decisions, and future direction.

## Sentient Connection

Sentient operations live in:

- `Agent.md`
- `Orchestration.md`

These files describe agent rules, validation duties, branch discipline, and task history.

## Review-Sensitive Areas

Do not change these without review:

- Next.js application routes and product UX.
- Dependency and script metadata.
- Lockfiles.
- Deployment behavior.
- Environment files and secrets.
